# MedRAG Textbooks Corpus

Medical textbook corpus, used to supplement `datasets/guidelines/` in the RAG
knowledge base. Part of the MedRAG project's four corpora — only this one (not
`MedRAG/pubmed` at 70GB or `MedRAG/wikipedia` at 45.7GB) is in the core backbone;
see `docs/13_Datasets.md` corrections section for why the other two were skipped.

- Size: 212 MB
- HF repo: `MedRAG/textbooks`

Fetched by `scripts/fetch_datasets.py --dataset medrag-textbooks`.
