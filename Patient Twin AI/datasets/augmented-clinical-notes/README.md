# Augmented Clinical Notes (synthetic)

Synthetic clinical notes (structured from PMC case reports), MIT licensed, no
credentialing. Bonus find during dataset research — not real patient data, so it's
useful as an open, no-license-needed stand-in for `doc_coding_service` dev/testing
while `datasets/MIMIC-IV-Notes/` credentialing is pending. Not a substitute for real
EHR notes at eval/release-gate time.
