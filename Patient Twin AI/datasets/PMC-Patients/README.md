# PMC-Patients

Open clinical-case-summary dataset (patient summaries extracted from PubMed
Central case reports). Used as an open stand-in / supplement for clinical-notes
retrieval and coding while MIMIC-IV Notes credentialing is pending.

- Size: 1.38 GB

  leave a research/eval context.
- HF repo: `zhengyun21/PMC-Patients`

Fetched by `scripts/fetch_datasets.py --dataset pmc-patients`.
