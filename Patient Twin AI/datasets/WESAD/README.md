# WESAD (Wearable Stress and Affect Detection)

Chest (RespiBAN) + wrist (Empatica E4) recordings, 15 subjects, stress/amusement/
neutral conditions. Used for baseline/deviation-engine validation on physiological
stress signals — see `docs/13_Datasets.md` for why this is the "best" pick for the
wearables category (paired with `datasets/PPG-DaLiA/`).

**Not committed** — download it yourself and drop the per-subject files here.

(`eti.uni-siegen.de/ubicomp/home/datasets/icmi18`) is mid-redesign with no working
direct-download link as of 2026-07-03 — see the corrections section in
`docs/13_Datasets.md`. Before scripting a pull, confirm a current mirror (it is
